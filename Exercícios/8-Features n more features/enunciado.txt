Aqui podemos implementar as novas features do MVP!!

IDEIAS
1-Barra de favoritos
2-Carrinho de compras

IMPLEMENTAÇÃO
1-Barra de favoritos
  a) Colocar o botão na navbar e em cada peça
    i) baixar img
    
    ii) Colocar no navbar

    iii) Criar a rota

    iv) Colocar a opção de fav em cada peça

  b) Plugar uma view
    i) criar a view favorite

    ii) criar uma view unfavorite

  c) Lógica por trás do Model
    i) Criação de uma linha no Model Favorite apertando o butão de favoritos

    ii) Remoção dessa linha assim que apertamos o botão de unfavorite


2-Carrinho de compras
  a) Colocar o botão do carrinho no nav bar e ir para uma rota especial
    i) baixar img

    ii) Criar a rota cart

    iii) Apontar a img para esta rota

  b) View cart
    i) Criar a view deste cart

  c) Model Cart
    i) Criar o model Cart

  d) Controller Cart
    i) Quando clicamos em "save stock", temos que salvar a camisa no carrinho
